Loot

Loot items are randomized pieces of equipment that are rewarded over the course of a run.
Items are obtained from Data storages. These will contain 3-7 items depending on party size. Only one item can be selected per player; if multiple players select the same item, the Cache will select a random player unaffected by luck. The other players can pick another available item.
Each item belongs to a Loot Set. Items belonging in to the same Loot Set share a theme, and often have similar mechanics. Some Loot Sets are unlocked through completing specific goals or defeating a certain number of bosses.

Data storages

  • There are five different types of DataStores: FDD, ROM , HDD , RAM, SSD, and CLOUD
  • What storage type you get is completely random. There are ways to spend in-game currency at the Shop to select a specific Storage type in the next arena
  • Each Storage type will hold unique loot sets with some sets overlapping multiple storages
    • There are special types of Storages that will have a unique predetermined selection of items unlocked via [CORRUPTED] mode only acessible from overriding at the Shop
    • E.g. VRAM, CD-ROM, Blu-ray, M.2 NVMe

Party Size on Loot Pool Size

The first stage The Desktop's two DataStores will always have 6 randomly generated loot to pick from, The next Storages will contain a specified number based on the party size:

Team Size 1 2 3 4 5 6
Pool Size 3 4 4 5 6 7
